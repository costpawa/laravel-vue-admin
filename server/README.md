## Laravel Admin Api

Cms APIs with Laravel

## Setup

```bash
# install dependencies
$ composer install

# copy .env.example file into .env then update your database name and credentials
$ cp .env.example .env

# run migrations
$ php artisan migrate

# serve at localhost:8000
$ php artisan serve

# your apis will be served under http://localhost:8000/api/v1
```
